⚙️ Prerequisites

Before using these workflows, ensure you have:

✅ n8n installed (local or cloud)
🔑 Required API keys (varies per project)
🌐 Internet access for external integrations
🚀 Getting Started

Clone this repository:

git clone https://github.com/vaishnaviboobesh/n8n-project.git
Open your n8n instance
Import a workflow:
Go to Workflows → Import from File
Select the workflow.json file from any project folder
Configure credentials:
Add API keys (OpenAI, Telegram, etc.)
Update environment-specific values
Activate and run the workflow
🔐 Credentials & Security
❗ API keys and secrets are NOT included in the exported workflows
🔒 Always configure credentials securely in n8n
🛑 Never commit sensitive information to this repository
