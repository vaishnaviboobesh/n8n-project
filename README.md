# 🚀 n8n Workflows Collection

Welcome to my n8n Workflows Repository!  
This repository contains a collection of automation workflows built using n8n, covering various real-world use cases like AI agents, integrations, notifications, and productivity automations.

---

## 📌 About This Repository

- 🧩 Each folder represents a separate n8n project/workflow  
- 📄 Every project includes its own dedicated README with:
  - Workflow description  
  - Setup instructions  
  - Required credentials  
  - Example usage  
- 🔄 Workflows are exported as `.json` files and can be directly imported into n8n

## 🚀 Getting Started

1. Clone this repository:

```
git clone https://github.com/vaishnaviboobesh/n8n-project.git
```

2. Open your n8n instance  

3. Import a workflow:
   - Go to **Workflows → Import from File**  
   - Select the `workflow.json` file from any project folder  

4. Configure credentials:
   - Add API keys (OpenAI, Telegram, etc.)  
   - Update environment-specific values  

5. Activate and run the workflow
